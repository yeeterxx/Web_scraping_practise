# Web scraper: quotes to scrape

This is a simple Python web scraper that collects quotes, authors, and tags from http://quotes.toscrape.com/ and saves them to a JSON file.

## Features
- Scrapes multiple pages automatically
- Extracts quote text, author, and tags
- Saves data to `quotes.json`
- Clean and readable JSON format

## Requirements
- pip install -r requirements.txt

## How to Run
1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Run the scraper: `python scraper.py`
4. Check `quotes.json` for output
