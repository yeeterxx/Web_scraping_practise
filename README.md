# Web scrapers

This is a simple Python web scraper that collects data from different websites and saves them to a JSON file.

## Features
- Scrapes multiple pages automatically
- Extracts quote text, author, and tags
- Saves data to 
- Clean and readable JSON format

## Requirements
- pip install -r requirements.txt

## How to Run
1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Run the scrapers
4. Check the given json for output
